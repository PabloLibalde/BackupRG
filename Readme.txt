# Apresentação do Projeto

## Objetivo
Feito, mais para usuarios do Windows. Este projeto tem como objetivo principal realizar o backup e organização de arquivos importantes, garantindo segurança e acessibilidade.

## Estrutura
- **Diretório Principal**: `/g:/Meu Drive/RG/D - Projetos/BackupRG/`
- **Arquivo de Documentação**: `Readme.txt` (este arquivo)

## Funcionalidades
1. Backup automático de arquivos (Pelo Agendador de Tarefas do Windows).
2. Organização por categorias e datas.
3. Logs detalhados para auditoria.

## Benefícios
- Segurança dos dados.
- Facilidade de acesso e recuperação.
- Histórico completo de alterações.
